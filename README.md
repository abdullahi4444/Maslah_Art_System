# 📂 Project Structure — Maslah Arts 🎨

This structure outlines every file, its purpose, and the responsible team members. Each section is clearly labeled to ensure everyone understands their role in the system.

```
MaslahArts/
├── index.php                  # Homepage (Landing page by Maymun)

├── Includes/                  # Common components (Abdiqadir Aydid)

│   ├── footer.php             # Footer section

├── Blogs/                     # Blog & Shop (Hafsa)

│   └── shop.php               # Artwork shop
```

## 🤝 Collaboration Approach

- Each team member is responsible for the development and debugging of their assigned folder/pages.
- Code reviews and feature integrations are done via regular check-ins and Git version control.
- We ensure a clean, modular codebase for scalability and future development.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Design Tools**: Figma (for UI planning)
