# 🎨 Maslah Arts

Welcome to the **Maslah Arts** system repository. This project is built by a team of dedicated developers and designers collaborating to create an elegant platform that showcases art through multiple categories such as digital prints, blogs, and student portfolios.

## 🤝 Collaboration Approach

- Each team member is responsible for the development and debugging of their assigned folder/pages.
- Code reviews and feature integrations are done via regular check-ins and Git version control.
- We ensure a clean, modular codebase for scalability and future development.

---

## 🛠️ Tech Stack

- **Frontend**: HTML, CSS, JavaScript
- **Backend**: PHP
- **Database**: MySQL
- **Design Tools**: Figma (for UI planning)
